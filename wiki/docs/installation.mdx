# Installation

Here are the following ways to install CoreCraft depending on your launcher. We recommend using Modrinth App or Prism Launcher to install CoreCraft for easier modpack installation and updates.

### Modrinth App

1. Install Modrinth App: https://modrinth.com/app
2. Go to "Discover Content" tab
3. Search for CoreCraft and press "Install"

### Prism Launcher (or MultiMC)

1. Install Prism Launcher: https://prismlauncher.org/
2. Press "Add Instance"
3. Go to the "Modrinth" tab and search and filter for CoreCraft

### Minecraft Launcher

We do not recommend this, there is lots of manual work which is prone to user error and it also takes some setup to configure MacOS microphone permissions.

1. Install Minecraft Launcher: https://www.minecraft.net/en-us/download
2. Install Fabric: https://fabricmc.net/use/installer/
   - You might need to install Java: https://docs.fabricmc.net/players/installing-java/
3. Download the modpack from our Modrinth page: https://modrinth.com/modpack/corecraft
4. Convert .mrpack to .zip using https://jamie.codeberg.page/mrpack-to-zip/@master/
5. Extract the zip file and copy the files into your game directory: https://minecraft.wiki/w/.minecraft
6. Go to the "Installations" tab in Minecraft: Java Edition, make a new installation, set your version to the corresponding fabric loader, and set your game directory

# Updating the Modpack

### Modrinth App

The linked modpack will display an "Update" button

### Prism Launcher

- Select the instance
- Click on "Edit"
- Go to the "Modrinth" tab
- Update to version
- Press "Update Pack"

### Minecraft Launcher

Backup your worlds, screenshots, and world-map data. Then go through the reinstallation process.
